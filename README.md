# LooooootyWeb (Staff Panel)

Initial scaffold for your web panel.

## 1) Install

```bash
cd /Users/looooooty/Documents/LooooootyWeb
npm install
```

## 2) Create env

```bash
cp .env.example .env
```

## 3) Run

```bash
npm start
```

Open: http://localhost:4000
